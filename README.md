## ⏱️ 1일 1문제 루틴
1. **[Step 1] 분석 (15-20분):** 문제 분석 및 슈도코드(Pseudo-code) 설계
2. **[Step 2] 구현 (30-40분):** 실제 코드 구현 및 예외 처리
3. **[Step 3] 피드백 (20분):** 복잡도 계산 및 타인의 효율적인 코드와 비교

### 💡 나만의 Rule
- **Time Limit:** 한 문제에 1시간 이상 실마리가 보이지 않으면 솔루션 참고 후 분석.
- **Complexity:** 제출 전 반드시 **시간/공간 복잡도**를 계산하는 습관 들이기.
- **Review:** 주말에는 주중 오답 문제 및 유사 유형 다시 풀기.
